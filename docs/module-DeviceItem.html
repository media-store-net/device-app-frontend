Module: DeviceItem

DeviceItem is a Component to reflect functionality of one device object

Props

Name Type Default value Required? Description
id number - Yes id of the item, this prop is required
companie object - Yes represent the companie / customer of the device, this prop is required
part object - Yes represent the doctype of the device, this prop is required
sn string - Yes represent the series number of the device, this prop is required
files Array [] No Files Array include all the Files of one component
isAdmin Boolean false No isAdmin prop allows to show or hide admin-specific buttons and actions

Data

Name Type Default value Description
icons object {mdiQrcode, mdiPencil, mdiDelete, mdiFile} used MDI Icons inside the component https://materialdesignicons.com
showFiles Boolean false this property allow to show and hide Files-Component

Computed

Name Type Description
doctypes Array.<object> the doctypes property will be loaded from the VUEX store via mapGetters() function

Events

Name Payload Type Description
gen-qr number Emit 'gen-qr' event with the given id of device if the generate qr button clicked
edit-device number Emit 'edit-device' event with the given id of device if the edit-button clicked
delete-device number Emit 'delete-device' with the given id of device event if the delete-button clicked
